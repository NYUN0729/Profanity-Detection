# AI 기반 영상/오디오 욕설 자동 탐지 시스템

## 프로젝트 소개

본 프로젝트는 영상 또는 오디오 콘텐츠에 포함된 욕설, 비속어, 부적절한 표현을 자동으로 탐지하기 위한 AI 기반 음성 분석 시스템입니다.

영상 콘텐츠나 오디오 파일에서 음성을 추출한 뒤, 화자 분리 모델과 음성 인식 모델을 활용하여 발화자별 대화 내용을 텍스트로 변환합니다. 이후 GPT 기반 텍스트 분석을 통해 욕설 및 비속어가 포함된 구간을 자동으로 탐지하고, 시간 정보와 화자 정보를 함께 출력합니다.

본 시스템은 콘텐츠 편집, 욕설 필터링, 자막 검수, 온라인 플랫폼 모니터링 등에 활용할 수 있습니다.

---

## 프로젝트 배경

미디어 산업이 발전하면서 영상, 오디오, 실시간 스트리밍 콘텐츠 내 부적절한 언어 표현을 자동으로 탐지하는 기술의 필요성이 증가하고 있습니다.

특히 욕설과 혐오 표현은 단순한 언어 문제가 아니라 사회적 갈등, 정서적 불쾌감, 온라인 커뮤니케이션 환경 악화로 이어질 수 있습니다.

기존 욕설 탐지 연구는 주로 텍스트 채팅이나 이미지 내 텍스트 탐지에 집중되어 있었으나, 본 프로젝트는 영상 및 오디오 콘텐츠를 대상으로 음성 인식, 화자 분리, GPT 기반 의미 분석을 통합한 자동 욕설 탐지 시스템을 구현했습니다. :contentReference[oaicite:0]{index=0}

## 주요 기능

- 오디오 파일 입력
- 영상/오디오 내 음성 분석
- pyannote.audio 기반 화자 분리
- Whisper 기반 음성 텍스트 변환
- 발화 시간 구간 추출
- 화자별 발화 내용 정리
- GPT 기반 욕설 및 비속어 탐지
- 욕설 단어별 등장 횟수 분석
- 욕설이 등장한 시간 및 화자 정보 출력
- 분석 결과 텍스트 파일 저장
